# 爬虫训练营排期看板

展示每个在营营期当前处于第几周、班主任本周要做什么。纯静态页面，随仓库发布到 GitHub Pages，访问路径为 `/camp-schedule/`。

## 营期管理（页面上直接操作）

- **添加营期**：点右上角「＋ 添加营期」，填期数和开营日期（开营典礼直播那天，一般是周日）即可生成整期排期
- **长假跳过**：添加/编辑时点选碰上五一、十一、春节等长假的课程周，该周变为「休息周」，后续课程整体顺延一周；正常情况不用选
- **编辑 / 删除**：每个营期卡片左上角有对应按钮
- **数据存哪**：保存在浏览器 localStorage，同时实时同步到网址 `#` 后面。点「复制分享链接」把网址发给班主任，对方打开就是同样的营期配置。改动后记得重新发一次链接

URL 格式：`#期数:开营日期:跳过周.跳过周`，多期用逗号分隔，例如 `#5:2026-06-07:6,6:2026-08-30`。

## 周期模型

| 周段 | 时间 | 说明 |
|---|---|---|
| 开营周 | 开营直播当周（周一~周日） | 建群建课、入学档案、分组，周日开营直播 |
| 第 1~12 周 | 开营后 12 个自然周（周一起算） | 常规督学 SOP |
| 第 4 周 | — | 叠加：工信部考试报名 |
| 第 5 周 | — | 叠加：工信部考试周 + 接单测评拉群 |
| 第 12 周 | — | 结营周（周日结营直播） |
| 休息周 | 勾选的长假周 | 无课程更新，课程顺延一周 |

支持任意多期并行，宽屏下一屏并排展示三期；已结营的营期自动折叠为一行。顶部「今日待办」聚合所有进行中营期当天的任务（「全周/日常」类任务不列入，见各营期卡片）。

## 执行追踪（防遗漏）

- **任务勾选**：每条任务前有复选框，勾掉表示完成；周面板和今日待办联动，每周显示完成进度（如 已完成 3/26）。勾选状态存在各自浏览器，不随分享链接传播
- **逾期标红**：本周已过去的天里没勾掉的任务，在今日待办顶部标红为「逾期 · 周几」；查看过去的周时逾期任务同样标红
- **时间标签**：有硬时间点的任务（15:00前、17:00 等）显示红色时间标签，今日待办按时间排序（`time` 字段）
- **持续跟进**：寄证书、寄奖品这类没有固定日期的跨周任务（`follow: true`），到期后会一直出现在今日待办底部的「持续跟进」区，勾掉才消失——即使该期已结营

## SOP 内容更新

每周具体工作内容在 [data.js](data.js) 里（`SOP_OPEN` / `SOP_REGULAR` / `SOP_EXTRAS` / `SOP_CLOSING` / `SOP_REST`），任务格式：

```js
{ day: 1, t: '任务标题', d: '补充说明（可选）', link: 'duxue', tag: 'exam' }
// day: 1~6=周一~周六，0=周日，'all'=全周/日常
// link: 对应 LINKS 里的键，会渲染成飞书链接
// tag: 'exam'/'close'/'open'，用于特殊任务打标
```
